## Compilando o jogo

Para compilar o jogo e necessaria a instalaçao das seguintes bibliotecas:

- libsdl2-dev
- libsdl2-image-dev
- libsdl2-mixer-dev
- libsdl2-ttf-dev

Em seguida basta utilizar o comando "make" para compilar um excecutavel.
